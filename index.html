About

Hi I'm Lucci, a front end developer and artist from Southern California. I'm passionate about coding clean, functional and elegant solutions in the form of websites. Currently I work as a freelancer in web development/design and as a graphic designer for an educational YouTube channel with over 150,000 subscribers.

I've been messing around with HTML and CSS since the Myspace days; I was obsessed with page customization. At the end of my senior year in high school, I decided to buckle down and learn how to program. I started out with Python on Codecademy and Coursera, then later took different college courses in C, Java, Ruby, Ruby on Rails and Javascript. However, those courses didn't provide enough depth or practical knowledge for me to feel confident in making my own projects; I didn't know where to start!

After a couple of hiatuses from coding (that were spent working restaurant jobs and traveling), I decided it was best for me to narrow my learning to front end development. The artistic design aspects of web development have always appealed to me most; I just really enjoy making things look cool. Since then, I've been enthusiastically upgrading my skills with freeCodeCamp, Udemy courses and a few trusting website clients.

Portfolio

Solario

Solario is a vanilla JS web app that helps the user protect their skin from the sun's harmful rays. This was the first time I used WebPack and Babel in my build process; tricky to get the hang of but it really facilitated my development workflow. The app uses the Geolocation Web API along with the SunCalc and Moment.js libraries to display the sun's altitude at the user's current position and time. Depending on the altitude, it also displays advice on whether to avoid sun exposure and facts about UV rays and the sun. As someone who is interested in skincare and health, this app is very useful to me; I built this for myself. :)

Built using: Javascript, Geolocation Web API, SunCalc library, Moment.js, HTML5, CSS3, Sass, npm, WebPack, Babel

Christie Audio

Similar to my previous project, I helped set this client up with web hosting and a domain on SiteGround (great customer service and prices, I highly recommend them). I created a child theme from a cool template I found on Wordpress.org, but unlike my last client's theme, this one was very stubborn and difficult for me to modify. I had to implement many workarounds; most of my design was done in the visual customizer in plain CSS, the audio player I created was separated into different parts (an HTML widget, the Additional CSS section, and a Javascript plugin), and the gallery is really just a blog post preview slideshow that I beat into submission. In retrospect, I definitely should have attempted to recreate the theme from scratch in order to have more control. Photoshop was used to create the logo in the top left corner, and Lightroom to edit his photos uniformly for the slideshow gallery. The client seems very satisfied with the results.

Built using: HTML5, CSS3, Wordpress, jQuery, Photoshop, Lightroom

Twitch Status Viewer

This freeCodeCamp project was completed in late 2017. I utilized a JSON call to retrieve objects from the Twitch API, they weren't too difficult to work with thanks to jQuery. I learned more about background attributes in CSS, specifically the blend mode. In making it responsive, I used art direction (just recently learned the term) to display the logo differently on smaller devices. Twitch is a great platform and I often watch streamers, so it was cool to make something that I would use for myself. I still love the design and color palette of this project; I think the result is super cute.

Built using: HTML, CSS, jQuery, JSON
